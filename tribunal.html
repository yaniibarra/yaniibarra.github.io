The system created for the supreme court In the elaboration of the web system for the registration of licenses, they were carried out in a practice leaves requested by court employees who are absent for a period of time.

Login

form with user validation

Employees Registration

A form was designed for the registration of employees that counts the TSJED inserting the area where you are, your status, your category (type of position) you occupy and at the end the name complete.

Licenses Registration

the form is displayed allows the registration of requested licenses by the TSJED workers themselves, which consist of verifying the type of license and the days allowed to comply with the organic law of the state of Durango.

Superiors

a form was designed to register the SUPERIOR with the following data (employee name, boss name and status). In the field of employee will display the name of the worker that is related to the superior.

Trades

A form was made in OFFICES to register your number, type of license, the name and status. Each trade made has the name of the employee, the name of the boss to be signed by the judges

Proceedings

a form is displayed with the following fields (ID record, name of record and the date on which the record is drawn up). The minutes are a report of the licenses requested by each TSJED worker.

Updates

Implemented a table in HTML5 and CSS to display the data saved from the database of data, the table has a button where the administrator, when clicking, will appear a form with all the data entered, in this way the person will edit the data that you want to change.

Queries Employees

In the PERMISSION queries option, the information of the fields is displayed permit id, permit type, start date, end date and status that an employee has requested displaying the permission key information.

Status Changes

The following table only shows the number of all the workers who have been granted a license, its name and status which consists of only editing the status field to modify the person to be unsubscribed.

Permit status

In the CHANGE PERMISSIONS menu, it shows the data of the licenses granted. The data are the permit number, status and its type of license, the observed table will only change if the license is registered

The TSJED was important in developing this system because previously there was no clear organization of all the information, they also did not have the information at make a request for a leave by the employee and the requested days of that way they did not know if the person had already applied for some type of license.
